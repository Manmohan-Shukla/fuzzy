# fuzzy-waffle-cli-github
# Fuzzy Logic Experiments with NumPy

This repository contains my personal learning experiments and implementations related to **fuzzy logic** concepts, built primarily using **NumPy**.

The focus is on understanding core ideas such as:
- Membership functions
- Fuzzy sets and operations
- Fuzzification and defuzzification
- Simple rule-based fuzzy systems
- Basic applications (e.g., decision making, control systems)

All code is written in Python using Jupyter Notebooks for interactive exploration and a few supporting .py scripts.

## What's Inside

- **phase-1.ipynb** → Initial exploration of fuzzy sets and basic membership functions using NumPy arrays  
- **phase-2.ipynb** → Fuzzy operations (AND, OR, NOT, implication, aggregation)  
- **phase-4.ipynb** → More advanced examples (possibly Mamdani or Sugeno style inference)  
- **kase-3.ipynb** → Case study / special experiment  
- **cli.py** → Simple command-line interface (possibly for running fuzzy inference)  
- **config.py** → Configuration utilities  
- **github_api.py** → Helper script (maybe for repo management or unrelated experiment)  
- Various .npy files → Saved NumPy arrays used in examples (membership functions, test data, etc.)

## Technologies Used

- Python 3
- NumPy (core library for array operations and fuzzy calculations)
- Jupyter Notebook (for interactive development and visualization)
